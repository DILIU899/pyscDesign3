pyscDesign3.tools.get_bpparam.get_bpparam

pyscDesign3.tools.get_bpparam.get_bpparam = <function get_bpparam>[source]

Get your parallelization parameters robject

Check R function BiocParallel::MulticoreParam and BiocParallel::SnowParam for more informarion on how to set the parameters.

Arguments:

name:

str The type of your selected parallel parameter. If windows, choose SnowParam. If linux or mac, choose MulticoreParam or SnowParam.

show:

bool (default: True) Whether to print the constructed onject information on the screen.

Output:

rpy2.robjects.methods.RS4

A R class specifying the parrallel parameters.

Combined use with @parallelization = bpmapply.
